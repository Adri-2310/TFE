## Cas d'Utilisation Admin Secrétariat

> **Note importante sur le rôle AdminSecrétariat :**
> Le rôle AdminSecrétariat est attribué **automatiquement lors de l'inscription** du propriétaire/patron d'un secrétariat social sur la plateforme. Une fois inscrit, l'AdminSecrétariat principal peut créer d'autres AdminSecrétariat (employés) pour son équipe. Tous les AdminSecrétariat d'un même secrétariat ont accès aux mêmes fonctionnalités, mais seul le propriétaire (marqué `is_main_admin: true`) est identifié comme compte principal.
