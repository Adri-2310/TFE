💡 Comment utiliser

1️⃣ Visualiser

Cliquez sur une maquette pour l'ouvrir en plein écran. Testez les interactions (modals, tabs, filtres).

2️⃣ Convertir en PNG

F12 → Imprimer (Ctrl+P) → Enregistrer en PDF/PNG pour le deck de présentation client.

3️⃣ Adapter au dev

Copier le code HTML → Adapter en React + Next.js + Shadcn/UI + TweakCN.
